####Summary
rkt is a docker competitor. Including here because it runs on a chromebook with crouton.

####Installation

```
cd
mkdir rkt_download
cd rkt_download
git clone https://github.com/coreos/rkt.git
cd rkt/scripts
sudo ./install-rkt.sh
```
